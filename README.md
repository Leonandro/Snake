# Compiling

1 - `mkdir build && cd build`
2 - `cmake -G "UnixMakefiles" ..`
3 - `make`


# run the game 
`./game`
or
`./game <Args>` 

# Argurments
To write the path of the initial data please: use the `-a` flag
**EX** `./game -a ../inlude/game1.dat`
If you loose the '-a' flag, the game will not use your path.
